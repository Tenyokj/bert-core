**Glossary**

**Admin Slot (EIP-1967)**  
Storage slot where proxy admin address is stored.

**Implementation**  
The logic contract behind a proxy. Upgrades change this address.

**ProxyAdmin**  
Contract that owns the upgrade authority for a proxy.

**Role**  
Permission granted via `RolesRegistryUpgradeable` to a contract or user.

**Voting Round**  
A fixed period in which ideas are voted on.

**Idea Status**  
The lifecycle stage of an idea (Pending, Voting, WonVoting, Funded, Rejected, Completed).

**Stake**  
Tokens locked to vote in a round.

**Grant**  
Funds distributed to the winning idea author.

**Pausable**  
A safety mechanism that halts actions in a contract.

**Upgradeable**  
A contract pattern where logic can change without moving state.
